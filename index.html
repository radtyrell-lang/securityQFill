FAQ

What does “grounded-only” mean?

We only answer questions backed by a verified fact — from your GitHub scan, your uploaded documents, or your own attestations. When there’s no source, we don’t generate anything. Instead we produce an honest, professionally worded gap response you can submit as-is. That refusal is the feature, not a limitation.

What if I don’t have a SOC 2 or a pen test?

That’s exactly who this product is for. Those questions become honest gap responses with specific next steps — for example, “We have not completed a SOC 2 audit; it’s on our roadmap as a Series A company.” Enterprise procurement handles honest gaps far better than fabricated controls that fall apart under a follow-up question.

How is this different from just using ChatGPT?

ChatGPT generates plausible-sounding answers with no source. In a security review, submitting a fabricated “yes” to a control you don’t have is a misrepresentation liability. SecurityQFill refuses to answer without a verified source, cites where each answer came from, and gives you honest gap responses everywhere else — plus a remediation report ChatGPT can’t produce.

Is my source code safe?

Yes, by design. The planned GitHub scan reads only specific named files — branch protection settings, CI workflow configs, dependency manifests, and Terraform / IaC files. Never a full clone, never arbitrary source. All calls are designed to go directly from your browser to GitHub; your code and questionnaire documents are never routed through our servers. The exact list of files the scan can read will be shown to you before you authorize access.

What does the gap analysis include?

A list of every unanswered question paired with a specific remediation recommendation and a rough time and cost estimate — so before the deadline you know exactly what to work on, what’s a quick $0 fix, and what (like a pen test) simply can’t be done in time and should be disclosed honestly instead.

What’s my LLM cost?

Under $1.50 even on a 300-row questionnaire at premium model pricing — often a fraction of that on a lighter model. You bring your own key, so you pay your provider directly at cost. We never see your data, and we never mark up your inference.

What questionnaire and question formats are supported?

XLSX, CSV, and DOCX — the formats procurement actually emails, including SIG Lite, CAIQ, and freeform vendor forms. We fill answers back into the original file and preserve its formatting, and always confirm the column mapping with you before filling.

Which LLM providers can I use?

v1 targets Anthropic (Claude) using your own API key. Additional providers (OpenAI, Azure OpenAI, Google Vertex) are planned. If you have an existing enterprise LLM contract, the goal is that you can bring it.

Is there a Chrome extension?

Not for v1. A Chrome extension for auto-filling web-based vendor portals is planned for a later version. v1 focuses on the emailed XLSX / DOCX / CSV attachment that pre-SOC2 founders actually receive today.

What if something goes wrong?

If processing fails for any reason, you get a full refund. No questions asked.
